# MIDAS API 教学手册

> 面向工程师的 API 入门教程 —— 不需要编程基础，只需要懂工程。

## 这是什么？

MIDAS API 是一套基于 HTTP 协议的接口，让你可以通过发送 **JSON 数据** 来操作 MIDAS 结构分析软件（Civil、Gen 等）。简单说：**你不用点鼠标，用代码发指令就能完成建模、分析、取结果。**

## 你可以用 API 做什么？

| 功能类别 | 说明 | 对应章节 |
|---------|------|---------|
| DOC | 项目管理 — 新建、打开、保存项目 | [DOC.md](DOC.md) (11 个接口) |
| DB | 模型数据库 — 建节点、单元、材料、荷载... | [DB.md](DB.md) (225 个接口) |
| OPE | 操作命令 — 划分单元、计算截面特性... | [OPE.md](OPE.md) (13 个接口) |
| VIEW | 视图控制 — 选择、截图、显示控制 | [VIEW.md](VIEW.md) (7 个接口) |
| POST | 后处理 — 提取表格结果、内力、位移... | [POST.md](POST.md) (92 个接口) |

## 阅读顺序建议

1. **先读** [附录A：JSON 快速入门](appendix/json-basics.md) —— 5 分钟了解 JSON 是什么
2. **再读** [DOC 章节](DOC.md) —— 从最简单的接口开始，感受 API 怎么用
3. **然后读** [DB 章节](DB.md) —— 核心建模接口，工作中用得最多
4. **按需查阅** OPE、VIEW、POST 章节

7. 工程意义解释 — 为什么工程师需要这个功能

9. 批量生成案例 — 一次性生成多个调用
10. for 循环优化 — 用循环提高效率
